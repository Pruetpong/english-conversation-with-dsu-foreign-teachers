### API Endpoints

#### POST `/api/chat`

Streaming chat completions endpoint.

**Request:**
```json
{
  "messages": [
    {"role": "system", "content": "..."},
    {"role": "user", "content": "Hello"}
  ],
  "temperature": 0.7,
  "maxTokens": 800
}
```

**Response:** Server-Sent Events (SSE)
```
data: {"content":"Hello"}
data: {"content":" there"}
data: [DONE]
```

#### POST `/api/tts`

Text-to-speech generation endpoint.

**Request:**
```json
{
  "text": "Hello, how are you?",
  "voice": "onyx"
}
```

**Response:** `audio/mpeg` (MP3 binary)
